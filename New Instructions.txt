INITIAL SETUP:

1.)From main google drive page (where you found this file), click the drop-down arrow next to 'Lab Graphic Generator' in the top bar and hit 'Download'.

2.)Open downloaded compressed folder (Lab Graphic Generator-20200....). Extract the folder by hitting the "Extract All" button in the top right of the 'Compressed Folder Tools'
tab in Windows Explorer. Make sure to only run the program from the location to which it was extracted.

RUN PROGRAM:

1.)Place Excel (.xlsx) spreadsheet in main folder (lab_graphic_dist)
	a.)Ensure it has name format: "00-00-0000.xlsx"
	b.)Only one excel file should be present in the lab_graphic_dist folder at a time
	c.)Previous spreadsheets can be stored in "prev_sheets" folder (two examples and a blank template included)

2.)Run 'LabGraphicGenerator.exe' to generate corresponding graphics and output text file
	a.)The output graphics are in PNG format and will be saved in several 720x1280px frames based on the total number of cages
